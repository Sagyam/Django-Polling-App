## Getting Started

### Initial Setup
1. Make new virtualenv: 
 ```
 virtualenv env
 ```
2. Activate the virtualenv:
```
source env/bin/activate
```
3. Install Django: 
```
pip install Django
```
4. Edit for your timezone at:
```
docs/settings.py
```
5. Run the server 
```
python mange.py runserver
```
6. Open website 
```
http://localhost:8000/polls/
```
